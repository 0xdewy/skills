# Skills

A collection of Claude skills.

## Install

Add this repo as a tap to browse and install any skill:

```bash
hermes skills tap add iamky1e/skills
```

Or install a specific skill directly:

```bash
hermes skills install iamky1e/skills/skills/skill-creator
```

## Skills

| Skill | Description |
|-------|-------------|
| [skill-creator](skills/skill-creator/) | Create, refine, and publish Claude skills |

## Usage

After adding the tap:

```bash
# Search available skills
hermes skills search

# Install a skill
hermes skills install iamky1e/skills/skills/SKILL_NAME
```
